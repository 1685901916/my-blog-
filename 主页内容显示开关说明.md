# 主页内容显示开关使用说明

## 📌 功能说明

现在你可以通过配置文件控制主页显示哪些类型的内容！

## 🔧 配置位置

打开文件：`src/config.ts`

找到以下配置项（大约在第 44-49 行）：

```typescript
// 主页内容显示开关配置
homepageContent: {
    showBlogPosts: true,      // 是否在主页显示博客文章
    showMangaPosts: true,     // 是否在主页显示漫画内容
    showPortfolioPosts: true, // 是否在主页显示作品集内容
},
```

## ⚙️ 使用方法

### 1. 显示所有内容（默认）
```typescript
homepageContent: {
    showBlogPosts: true,
    showMangaPosts: true,
    showPortfolioPosts: true,
},
```
**效果**：主页显示博客文章 + 漫画 + 作品集

---

### 2. 只显示博客文章
```typescript
homepageContent: {
    showBlogPosts: true,
    showMangaPosts: false,
    showPortfolioPosts: false,
},
```
**效果**：主页只显示博客文章

---

### 3. 只显示作品集
```typescript
homepageContent: {
    showBlogPosts: false,
    showMangaPosts: false,
    showPortfolioPosts: true,
},
```
**效果**：主页只显示作品集内容

---

### 4. 显示博客 + 作品集（不显示漫画）
```typescript
homepageContent: {
    showBlogPosts: true,
    showMangaPosts: false,
    showPortfolioPosts: true,
},
```
**效果**：主页显示博客文章和作品集，不显示漫画

---

## 💡 重要提示

1. **修改后需要刷新浏览器**：修改配置后，等待几秒钟让开发服务器重新加载，然后刷新浏览器（按 `Ctrl+Shift+R` 强制刷新）

2. **各专题页面不受影响**：
   - `/manga/` - 漫画页面始终显示所有漫画
   - `/portfolio/` - 作品集页面始终显示所有作品集
   - `/portfolio/programming/` - 程序系页面始终显示程序系作品
   - `/portfolio/normal/` - 正常系页面始终显示正常系作品

3. **开关只影响主页**：这些开关只控制主页（`/`）显示什么内容，不影响其他页面

---

## 🎯 推荐配置

### 如果你想让作品集显示在主页：
```typescript
homepageContent: {
    showBlogPosts: true,      // 显示博客
    showMangaPosts: false,    // 不显示漫画（漫画太多会占用主页空间）
    showPortfolioPosts: true, // 显示作品集 ✅
},
```

### 如果你想保持主页简洁：
```typescript
homepageContent: {
    showBlogPosts: true,      // 只显示博客
    showMangaPosts: false,
    showPortfolioPosts: false,
},
```

## ❓ 常见问题

**Q: 为什么修改后没有变化？**
A: 请确保：
- 保存了 `src/config.ts` 文件
- 开发服务器正在运行
- 等待了几秒钟让服务器重新加载
- 使用 `Ctrl+Shift+R` 强制刷新浏览器

**Q: 作品集文件放在哪里？**
A: 
- 程序系作品：`src/content/portfolio/programming/`
- 正常系作品：`src/content/portfolio/normal/`

**Q: 如何确保作品集文件格式正确？**
A: 参考现有文件，确保 frontmatter 格式如下：
```yaml
---
title: 作品标题
published: 2025-01-01T00:00:00.000Z
description: 作品描述
image: ''
tags: ["程序系", "Python", "其他标签"]
category: 作品集
draft: false
---
```
