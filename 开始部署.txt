❓ 遇到问题？

常见问题解决：

1. GitHub 推送失败
   → 需要设置 Token
   → 脚本会自动提示

2. 服务器无法访问
   → 检查防火墙
   → 确认 Web 服务器运行

3. 文章不显示
   → 检查文件格式
   → 查看终端错误日志

详细解决方案请查看各个教程文档！
